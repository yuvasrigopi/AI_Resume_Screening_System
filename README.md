# 🤖 AI Resume Screening System

An AI-powered web application that automates the resume screening process by comparing multiple resumes against a job description. The system extracts candidate information, identifies skills, calculates ATS scores, ranks candidates, and generates downloadable PDF reports to help recruiters shortlist the best applicants efficiently.

---

##  Features

-  Upload and analyze multiple PDF resumes
-  Enter a custom job description
-  Extract candidate details (Name, Email, Phone, Education, Experience)
-  NLP-based skill extraction
-  ATS Score calculation
-  Resume-to-Job Description similarity analysis
-  Missing skills identification
-  Candidate ranking based on ATS score
-  Automatic PDF report generation
-  Responsive and user-friendly dashboard

## 🚀 Workflow

1. Upload one or more PDF resumes.
2. Enter the job description.
3. Extract resume text and candidate details.
4. Identify technical skills using NLP.
5. Compare resumes with the job description.
6. Calculate ATS score and match percentage.
7. Rank candidates based on suitability.
8. Generate downloadable PDF reports.

---

## Output

The system provides:

- ATS Score
- Resume Match Percentage
- Candidate Ranking
- Extracted Skills
- Missing Skills
- Candidate Status (Selected / Shortlisted / Rejected)
- Downloadable PDF Report
