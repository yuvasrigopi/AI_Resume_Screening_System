# AI Resume Screening System

## Features

- Upload Resume (PDF)
- Extract Resume Text
- Resume Skill Extraction
- ATS Score
- Resume Matching
- Missing Skills Detection
- Job Recommendation
- PDF Report Generation

---

## Install

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

Upload a resume PDF and view the analysis.
